# tot-app

Daily MLB betting picks with AI breakdowns, Stripe paywall, and automated Twitter posting.

**Live:** [tot-app.vercel.app](https://tot-app.vercel.app)

## Stack

Next.js · Supabase · Stripe · Anthropic Claude · Resend · Vercel

## Running locally

```bash
npm install
npm run dev
```

Requires `.env.local` with keys for Supabase, Stripe, Anthropic, Resend, and Twitter. Pull from Vercel:

```bash
vercel env pull
```
